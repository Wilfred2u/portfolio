1. My family tree

# My family tree
I've set up a database with about 250 ancestors of mine in Aldfaer, a Dutch ancestor-tree program. Then I've exported the database to a .gedcom-file. In Python, I've made a parser which parses the data of the .gedcom-file to multiple JSON-files. And finally, I made a JavaScript which reads the JSON-files and creates the whole family tree. Showing the scope of my family I found very important, to see all of my ancestors in the eighth generation properly, the HTML-file is forty eight thousand (48000!) pixels wide!
